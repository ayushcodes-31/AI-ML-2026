Functions
A block of reusable code that performs a specific task.
Helps avoid repeating code.
Parameters
Variables listed in a function definition.
They act as placeholders for values.
Arguments
Actual values passed to a function when calling it.
Return
Sends a value back from a function.
Ends the function execution.
Dictionaries
Store data in key-value pairs.
Mutable and unordered (insertion order preserved in modern Python).
Key
Unique identifier in a dictionary.
Used to access values.
Value
Data associated with a key.Lookup O(1)
Accessing an item in a dictionary or set is usually O(1).
Means constant time regardless of size.
Sets
Collection of unique elements.
No duplicate values allowed.
What is a Set?
Unordered collection of unique items.
Supports fast searching and insertion.
Why Use It for Duplicates?
Sets automatically remove duplicates.
Fast duplicate checking using O(1) lookup.
